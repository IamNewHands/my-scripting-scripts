## Features
- Probe formats first, then download with **yt-dlp** (video+audio in one extract when merging).
- **YouTube / Douyin / Xiaohongshu** login via WebView cookies when required.
- FFmpeg merge/verify, Photos/Files save, history, AVPlayer + open in other apps.
- TLS-friendly defaults; debug logs stay local (not published).
