## 功能
- 先探测格式，再用 **yt-dlp** 下载（需合并时 **一次** 拉取 video+audio）。
- **YouTube / 抖音 / 小红书** 可在需要时 WebView 登录取 Cookie。
- FFmpeg 合并校验、相册/文件保存、历史、AVPlayer 与「用其他 App 打开」。
- 默认兼容证书环境；运行日志仅存本机，**不上传仓库**。
